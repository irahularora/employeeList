# Development Setup Guide

To contribute to this project and set up a development environment, follow these steps:

## Prerequisites

- Node.js (v20.10.0 or later): [Download Node.js](https://nodejs.org/)
- npm (Node Package Manager): Included with Node.js installation
    -  npm install react-scripts
    -  npm install -g create-react-app

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/irahularora/employeeList
    ```

2. Navigate to the backend directory:

    ```bash
    cd backend
    ```

3. Install backend dependencies:

    ```bash
    npm install
    ```

4. Return to the project root:

    ```bash
    cd ..
    ```

5. Install project dependencies:

    ```bash
    npm install
    ```

## Running the Project

Run the following command to start both the frontend and backend servers:

```bash
npm run both
